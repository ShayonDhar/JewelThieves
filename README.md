# 💎 JewelThieves

[cite_start]A Java-based puzzle and strategy game where you must collect as much loot as possible and escape before the rival thieves do! [cite: 31, 86, 120]

[cite_start]This project is a digital recreation of the 1997 game *Jewel Chase* from the Microsoft Entertainment Pack[cite: 8]. [cite_start]It was developed as part of the CS-230 Software Engineering module[cite: 1, 2].

---

## 📖 About the Game

[cite_start]In JewelThieves, you navigate through levels made of square tiles[cite: 30, 38]. [cite_start]Your movement is restricted by color: you can only move to a tile if it shares at least one color with the tile you are currently standing on[cite: 42]. [cite_start]Your objective is to collect items, avoid traps and enemies, and reach the exit door before the timer expires[cite: 31, 120].

## ✨ Features

* [cite_start]**Three Unique Enemy Types:** Face off against Flying Assassins, Floor Following Thieves, and pathfinding Smart Thieves[cite: 62, 63, 74, 78].
* [cite_start]**Dynamic Items & Traps:** Utilize color-coded levers to open gates [cite: 105][cite_start], collect clocks for extra time [cite: 83][cite_start], and avoid proximity-triggered bombs that destroy paths[cite: 109, 110, 113].
* [cite_start]**Player Profiles:** Create a profile to track your name and the highest level you have unlocked[cite: 155, 156].
* [cite_start]**Save & Load Functionality:** Save your ongoing game state to a file, allowing you to resume your heist later[cite: 165, 166, 167].
* [cite_start]**High Score Tracking:** Compete for a spot on the top 10 high score table recorded for every level[cite: 158, 160].
* [cite_start]**Custom Level Parsing:** Levels are generated by reading custom ASCII-based text files [cite: 137] [cite_start]containing tile colors, NPC start locations, and level timers[cite: 140, 141, 142].

## 💻 Tech Stack

* [cite_start]**Language:** Java[cite: 188].
* [cite_start]**GUI Framework:** JavaFX (utilizing the Canvas class) and FXML[cite: 188].
* **Styling:** CSS.
* [cite_start]**Architecture:** Built strictly with standard Java SDK libraries, completely from scratch without external physics engines or game frameworks[cite: 189, 192].

## 🚀 Getting Started

### Prerequisites
* Java Development Kit (JDK) installed on your machine.
* JavaFX SDK configured for your environment.

### Installation & Execution
1. Clone the repository: `git clone https://github.com/ShayonDhar/JewelThieves.git`
2. [cite_start]Navigate to the `source` directory containing the application code[cite: 301].
3. Compile the Java files, ensuring your JavaFX modules are linked.
4. Run the main application class to launch the GUI.
5. [cite_start]Refer to the `README.txt` file within the source folder for any specific compilation flags or IDE instructions[cite: 302].

## 🎮 How to Play

* **Movement:** Use the directional keys to move. [cite_start]You can only step onto a tile if it shares a color with your current tile[cite: 42].
* [cite_start]**Looting:** Walk over coins, rubies, and diamonds to increase your score[cite: 100].
* [cite_start]**Bombs:** Stepping next to a bomb triggers a 3-second countdown[cite: 110, 111]. [cite_start]Get out of the blast radius before it destroys nearby loot and levers! [cite: 113]
* [cite_start]**Winning:** Collect all loot and levers, then reach the exit door before the time expires[cite: 120]. [cite_start]Beware: if a thief beats you to the door, you lose! [cite: 122]
