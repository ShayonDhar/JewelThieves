# 💎 JewelThieves
A Java-based puzzle and strategy game where you must collect as much loot as possible and escape before the rival thieves do!
This project is a digital recreation of the 1997 game *Jewel Chase* from the Microsoft Entertainment Pack.
It was developed as part of the CS-230 Software Engineering module.
---
## 📖 About the Game
In JewelThieves, you navigate through levels made of square tiles.
Your movement is restricted by color: you can only move to a tile if it shares at least one color with the tile you are currently standing on.
Your objective is to collect items, avoid traps and enemies, and reach the exit door before the timer expires.
## ✨ Features
* **Three Unique Enemy Types:** Face off against Flying Assassins, Floor Following Thieves, and pathfinding Smart Thieves.
* **Dynamic Items & Traps:** Utilize color-coded levers to open gates, collect clocks for extra time, and avoid proximity-triggered bombs that destroy paths.
* **Player Profiles:** Create a profile to track your name and the highest level you have unlocked.
* **Save & Load Functionality:** Save your ongoing game state to a file, allowing you to resume your heist later.
* **High Score Tracking:** Compete for a spot on the top 10 high score table recorded for every level.
* **Custom Level Parsing:** Levels are generated by reading custom ASCII-based text files containing tile colors, NPC start locations, and level timers.
## 💻 Tech Stack
* **Language:** Java.
* **GUI Framework:** JavaFX (utilizing the Canvas class) and FXML.
* **Styling:** CSS.
* **Architecture:** Built strictly with standard Java SDK libraries, completely from scratch without external physics engines or game frameworks.

## 🚀 Getting Started
### Prerequisites
* Java Development Kit (JDK) installed on your machine.
* JavaFX SDK configured for your environment.
### Installation & Execution
1. Clone the repository: `git clone https://github.com/ShayonDhar/JewelThieves.git`
2. Navigate to the `source` directory containing the application code.
3. Compile the Java files, ensuring your JavaFX modules are linked.
4. Run the main application class to launch the GUI.
5. Refer to the `README.txt` file within the source folder for any specific compilation flags or IDE instructions.

## 🎮 How to Play
* **Movement:** Use the directional keys to move. You can only step onto a tile if it shares a color with your current tile.
* **Looting:** Walk over coins, rubies, and diamonds to increase your score.
* **Bombs:** Stepping next to a bomb triggers a 3-second countdown. Get out of the blast radius before it destroys nearby loot and levers!
* **Winning:** Collect all loot and levers, then reach the exit door before the time expires. Beware: if a thief beats you to the door, you lose!
